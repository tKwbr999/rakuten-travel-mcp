### 3. 環境変数設定

`.env.example`ファイルを作成:

```
# サーバー設定
PORT=3000
NODE_ENV=development
LOG_LEVEL=info

# 楽天トラベルAPI設定
RAKUTEN_APP_ID=your_app_id_here
RAKUTEN_AFFILIATE_ID=your_affiliate_id_here

# MCP設定
MCP_SERVER_NAME=rakuten-travel-mcp
MCP_SERVER_VERSION=0.1.0
MCP_SERVER_DESCRIPTION=A Model Context Protocol server for Rakuten Travel API
```
